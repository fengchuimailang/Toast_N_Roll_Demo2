## 局内口味刷新与概率规则

| 1 局内口味刷新规则                                                        |
|:------------------------------------------------------------------|
| 进入任意已解锁区域的单局游戏时，固定刷新3种可制作的吐司口味，规则如下：                              |
| 1. 必出当前所选区域的专属口味                                                  |
| 2. 从玩家已解锁的所有其他口味中，随机抽取2种，共同组成本局3种可选口味                             |
| 3. 单局游戏内仅可制作这3种口味的吐司，不可切换其他口味                                     |
| 2 口味概率调整规则                                                        |
| 1. 专属口味初始出现概率统一为20%，                                              |
| 2. 其他口味(两种)为20% : 30% : 50%（价值越低出现概率越高）                           |
| 3. 可通过观看广告，或金币解锁，提升对应口味的出现概率，最高可提升至初始概率的2倍（即最高40%）40% : 30% : 30% |
| 3 小费动物概率提升规则                                                      |
| 可通过两种方式提升对应区域小费动物的出现概率：                                           |
| 1. 观看广告：将该区域小费动物出现概率临时提升至10%，仅生效于当前区域的下一局游戏                       |
| 2. 金币永久提升：花费金币，永久将该区域小费动物出现概率提升至15%                               |
