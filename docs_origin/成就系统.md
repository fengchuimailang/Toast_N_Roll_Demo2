## 四、等级与称号体系

共设置10个成长等级，以「累计完成吐司礼盒数量」为核心解锁指标，与核心玩法强绑定，每个等级对应专属称号与永久福利，阶梯式难度递增，激励玩家长线游玩。

| 等级序号 | 专属称号  | 解锁核心条件                                  |
| ---- | ----- | --------------------------------------- |
| 1级   | 烘焙小萌新 | 初始解锁，累计完成≥10个吐司礼盒                       |
| 2级   | 新手面包师 | 累计完成≥50个吐司礼盒，累计通关≥5局                    |
| 3级   | 实习烘焙师 | 累计完成≥150个吐司礼盒，累计通关≥15局                  |
| 4级   | 青铜烘焙师 | 累计完成≥400个吐司礼盒，累计通关≥30局                  |
| 5级   | 银牌烘焙师 | 累计完成≥800个吐司礼盒，累计通关≥60局，累计3星通关≥20次       |
| 6级   | 金牌烘焙师 | 累计完成≥1500个吐司礼盒，累计通关≥100局，累计3星通关≥50次     |
| 7级   | 烘焙大师  | 累计完成≥3000个吐司礼盒，累计通关≥200局，累计3星通关≥100次    |
| 8级   | 烘焙名家  | 累计完成≥6000个吐司礼盒，累计通关≥400局，累计3星通关≥250次    |
| 9级   | 烘焙宗师  | 累计完成≥10000个吐司礼盒，累计通关≥700局，累计3星通关≥500次   |
| 10级  | 吐司之神  | 累计完成≥20000个吐司礼盒，累计通关≥1200局，累计3星通关≥1000次 |

**补充说明**：累计完成吐司礼盒数量，指玩家所有对局中成功交付给顾客的礼盒总数，为等级提升的唯一核心指标。
