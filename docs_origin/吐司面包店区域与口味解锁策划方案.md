# 局内口味刷新与概率规则

## 关卡定义

1. 每个区域10个关卡，其中1到7关为固定关卡，第8-10关是循环关卡（eg: 7、8、9、10、8、9、10...）
2. 8-10关循环是为了让用户能够刷足解锁新区域所消耗的金币。
3. 区域只能顺序解锁

## 局内口味刷新规则

进入任意已解锁区域的单局游戏时，固定配置3种可制作的吐司口味，关卡设计规则如下：

1. 必出当前所选区域的专属口味，另外从玩家已解锁的所有其他口味中随机抽取1种，与原味共同组成本局3种口味
2. 单局游戏内仅可制作这3种口味的吐司，不可切换其他口味
