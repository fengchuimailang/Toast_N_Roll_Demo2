## Loading 页面

**当前实现基线**

- 加载页使用暖黄色烘焙店橱窗插画作为全屏背景，资源路径为 `assets/resources/ui/load_bg.png`。
- 中央保留独立的加载面板、假进度条与文案提示，避免背景图直接承载可读信息。
- 若背景图资源加载失败，允许回退到纯色底图，不能阻塞进入 Lobby。

## 🖱️ 交互规范

### 手势操作

| 手势 | 操作 | 反馈 |
|------|------|------|
| 点击 | 选中格子 | 高亮边框 |
| 滑动 | 交换相邻格子 | 交换动画 |
| 长按 | 显示食材信息 | 弹出提示 |

### 动画规范

| 动画类型 | 时长 | 缓动函数 |
|----------|------|----------|
| 格子交换 | 200ms | ease-out |
| 消除消失 | 150ms | ease-in |
| 食材下落 | 300ms | ease-in-out |
| 合成生成 | 250ms | ease-out |
| 礼盒飞行 | 500ms | ease-in-out |
| 弹窗出现 | 200ms | ease-out |

### 粒子效果

**消除特效**:
```
食材消除时:
└── 生成 8-12 个粒子
    ├── 粒子类型: 小方块/星星
    ├── 颜色: 食材对应颜色
    ├── 速度: 向外扩散
    └── 生命周期: 500ms
```

**合成特效**:
```
食材进阶时:
└── 生成光环效果
    ├── 中心: 进阶食材
    ├── 光环: 金色圆环扩散
    └── 时长: 400ms
```
