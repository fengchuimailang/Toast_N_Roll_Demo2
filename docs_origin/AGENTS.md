## 📌 关键设计约束

### 核心玩法约束（不可变更）
1. **格子布局**: 固定7×7正方形，初期教学关从3×2逐步解锁到7×7
2. **进阶链**: 严格6阶单向链，无反向降级、无随机变异
3. **补位规则**: 新食材从底部向上填充，上方食材保持原位
4. **交换规则**: 仅支持横竖相邻交换，禁止斜向

### 界面约束
- 竖版全屏布局
- 顶部20% / 中间70% / 底部10% 分区
- 无复杂弹窗遮挡核心操作区

### 扩展设计（可选实现）
- 口味标签系统（原味/抹茶/草莓）
- 小浣熊事件
- 14个区域关卡
- 5种道具
