# 《吐司面包店》核心道具简化表

| 道具名称     | 核心功能（不占用操作步数）                                                                 | 获取渠道（新手任务各给1个） |
|--------------|--------------------------------------------------------------------------------------------|----------------------------|
| 去除道具     | 点击清除任意1个食材，空位补齐                                                              | 广告兑换、成就解锁         |
| 洗牌道具     | 全局打乱格子内所有食材位置，完整保留食材阶数、口味标签，不触发合成操作，但有可交换的食材                     | 广告兑换、成就解锁         |
| 磁力道具     | 选中目标食材后，吸附最近的2个同级无口味食材直接合成；若当前不存在满足条件的目标，则道具不可用，并提示“当前没有可吸附的无口味同级食材” | 广告兑换、成就解锁         |

## 说明

- `无口味食材` 指原味食材，即未附加草莓、抹茶等口味标签的食材。
- 磁力道具不可对礼盒使用。
- 升级道具暂不纳入当前方案。
